Key Findings

What the Data Reveals

Analyzing IMDb's highest-rated films surfaces four clear patterns that separate beloved classics from the rest. Each finding below is supported by one of the visualizations in this dashboard.

1 2.8M

Audience Consensus at the Top

The Dark Knight and The Shawshank Redemption both lead with 2.8M reviews — suggesting that the highest-rated films also attract the broadest audiences, not just niche critics.

2 ~2000s

The Golden Era of Great Films

The years around 2000–2010 produced the highest concentration of top-rated movies (peak of 8 in a single year), pointing to a modern golden era for critically acclaimed cinema.

3 150+ min

Longer Films Rate Higher

Films exceeding 150 minutes consistently achieve higher average ratings than shorter counterparts — suggesting audiences reward ambitious, expansive storytelling.

4 9.2 avg

Mid-Century Films Still Shine

Older films from the 1940s–60s carry some of the highest average ratings, though with far fewer reviews — hinting at survivor bias where only the very best endure.

Insight

A Modern Renaissance in Cinema

The 2000s and 2010s dominate this chart. The peak year produced 8 top-rated films — more than any other single year in IMDb history. This era saw the rise of prestige television pushing filmmakers to deliver cinematic experiences worthy of the theater.

This pattern also reflects changing viewer behavior: more people rating films online means more data — but the quality concentration in this window is still notable compared to earlier decades.

→ If you're studying what makes a film great, the 2000–2015 window is the richest dataset to learn from.

Visualization 03

Average Ratings Over Time

Average Rating & Review Volume by Decade

Insight

Older Films, Higher Ratings — But Fewer Voices

Films from the 1940s–60s carry the highest average ratings (9.0–9.2), yet they have the fewest reviews. This is a classic case of survivorship bias — only the very best films from that era are still widely known and rated today.

Meanwhile, review volume skyrockets for post-2000 films as IMDb became the dominant platform for movie discussion. More voices leads to regression toward the mean, pulling average ratings slightly lower despite producing many acclaimed films.

→ Comparing ratings across decades requires accounting for participation rate, not just raw scores.

Insight

Longer Films Earn More Trust

There is a clear positive relationship between runtime and average rating among top IMDb films. Films over 150 minutes consistently rate higher than those under 120 minutes.

This likely isn't just about length — longer films tend to be passion projects with bigger ambitions, deeper character development, and more complex narratives. Audiences seem to reward that ambition. Notably, all five of the most-reviewed films are over 130 minutes.

→ Runtime correlates with rating, but causality likely runs through narrative depth and directorial ambition.

Data Pipeline

From Raw Data to Dashboard

Before any visualization, the raw IMDb dataset required meaningful cleanup. Here's the full workflow — from messy source data to polished dashboards in two different tools.

01

Raw Dataset (250 films, 7 columns)

The source data contained 250 IMDb top-rated films with fields for Ranking, Title, Rating, Number of Reviews, Year, Runtime (Minutes), and Content Rating (MBA Rating). Content ratings spanned multiple regional systems — R, PG-13, TV-MA, 16+, 18+, Unrated, Passed, Approved — reflecting inconsistent tagging across film eras and regions.

↓
02

Data Cleaning via Power Query

Before importing into Power BI, the dataset was cleaned using Power Query in Excel. This involved removing null and malformed entries, standardizing inconsistent content rating labels across different regional classification systems, ensuring correct data types for numeric fields like review counts and ratings, and filtering out rows with missing runtime or year values that would skew trend analysis.

→ Power Query was chosen because it creates a repeatable, auditable transformation pipeline — any future data refresh automatically applies the same cleaning steps.
↓
03a

Excel Dashboard

An initial dashboard was built directly in Excel using pivot tables and charts — covering total reviews by year, average rating by content rating category, average runtime by content rating, and number of films per content rating. This served as rapid exploratory analysis to identify which dimensions were worth investigating further.

↓
03b

Power BI Dashboard

The cleaned data was then imported into Power BI Desktop for a more polished, interactive dashboard. Power BI enabled richer visual storytelling — dual-axis charts comparing ratings and review volume over time, top-N filtering for the most reviewed films, and trend analysis across runtime segments. The Power BI report is the primary deliverable of this project.

About This Project

Background & Method

This dashboard was built as a university data analytics project using an IMDb dataset of 250 top-rated films. The goal was to practice the full analytics workflow — data cleaning, exploratory analysis, and visual storytelling — to uncover what attributes drive a film's rating and audience engagement.

The project demonstrates proficiency across the full BI stack: Power Query for data transformation, Excel for rapid exploratory dashboards, and Power BI Desktop for the final interactive report. Real-world analytics experience includes athletics ticket revenue analysis and operational dashboards (confidential).
